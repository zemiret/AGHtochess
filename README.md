# AGHtochess
Live deploy at [https://aghtochess.mionskowski.pl/](https://aghtochess.mionskowski.pl/)

## Installation

In order to deploy the application locally run:

```bash
docker-compose up -d --build
```

and navigate to `localhost:8080`
